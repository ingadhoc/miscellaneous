==============
Base Validator
==============

This module allows to set a python expression as value validator. This module is intended to be extended by others and should not be installed alone.

Installation
============

Only need to install module.

Configuration
=============

Do not need configuration

Usage
=====

To use this module, you need to:

#. Depend on this module
#. Create new records for new validations
#. Call validation from where you need
